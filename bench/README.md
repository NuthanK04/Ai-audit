# Benchmark Results

## Tokenizer Performance Benchmarks

This directory contains benchmark results and performance logs for the tokenizer audit.

### Contents

- `bench_log.csv` — Execution times and performance metrics
- `model_spec.md` — Tokenizer model specifications

### Usage

Benchmark results are used to track performance across different runs and identify regressions.

All benchmarks should be executed on the same hardware for fair comparison.
