# Corpus Sample

## Multilingual Text Samples

This directory contains sample text from different languages used for testing and demonstration.

### Language Coverage

- **English** — Sample English text for baseline comparison
- **Hindi** — Devanagari script sample
- **Kannada** — Kannada script sample  
- **Tamil** — Tamil script sample
- **Telugu** — Telugu script sample

### Usage

Samples are extracted from FLORES-200 for quick testing without downloading the full corpus.

Each sample file contains 10–20 representative sentences from the language.
